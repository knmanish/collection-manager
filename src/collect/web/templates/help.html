How to use collect

A two-minute tour. Everything lives on this device — nothing leaves it.

The three tabs

  • My Collection — a showcase of what you own: total value, a breakdown by category, and a picture for each piece. Click any piece to edit it.
  • Manage Collection — the workbench. Add a piece with the form, search and filter the table, and Edit / Remove existing ones. The right-hand panel is where you rename, add, or remove categories.
  • Wishlist — pieces you'd like to own. Each has a priority and an estimated price. Press Got it → to move one into your collection once you acquire it.

Adding & editing a piece

On Manage Collection, fill in the “Add a piece” form. Only a name and category are required. Leave the Image URL blank and collect will try to fetch a photo from the web automatically — if it picks the wrong one, open the piece and paste a correct image link to override it.

Money & currency

Each piece remembers the currency you entered it in. The USD EUR INR switch in the top-right converts every value on the page into that currency using live exchange rates (cached, so it still works offline).

Pictures

Images are pulled from the web based on a piece's brand and name. It's best-effort, so a guess can be off — the manual image link on the edit screen always wins. On My Collection, a “Fetch missing images” button appears when some pieces still need one.

Starting fresh

Your data lives in a folder in your home directory, separate from the app itself — so if you share the app with someone, they start empty. To wipe everything on this device and start over, use Clear all data at the bottom of the Manage tab.
